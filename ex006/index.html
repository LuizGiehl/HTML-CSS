Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag B (Não semântica)

Já aqui temos um termo em destaque usando a tag STRONG (semântica)

Itálico/Ênfase

Nesta frase temos um termo em itálico usando a tas i (Não semântica)

Já nesta frase temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos criar também um texto marcado com a tag MARK, e mudar a cor da marcação usando o STYLE da css

E temos outro texto marcado neste paragráfo

Textos grandes e pequenos

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como deletado para indicar que deve ser lido mas não considerado, usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para indicar que foi adicionado posteriormente

Também é possivel usar o sublinhado com a tag U (Não semântica)

Texto sobrescrito

Para inserir coisas como: 2x20+3 usamos a tag SUP

E para inserir coisas como: H2O usamos a tag SUB
